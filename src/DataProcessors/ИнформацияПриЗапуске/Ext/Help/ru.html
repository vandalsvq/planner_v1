В этой форме выводится справочная, нормативная и другая информация, поставляемая в составе программы.

Для открытия формы откройте Рабочий стол и вызовите команду "Информация при запуске" из группы "Информация" панели действий. При открытии сразу показывается одна из страниц дополнительной информации.

В командной панели формы размещены навигационные кнопки "Назад" и "Вперед", с помощью которых осуществляется переход по уже просмотренным страницам, а также пункты меню, соответствующие названиям разделов дополнительной информации. При выборе в командной панели раздела открывается список страниц дополнительной информации, относящихся к данному разделу. Страница показывается в форме при выборе пункта меню с названием страницы.

Текст страницы может содержать ссылки на другие страницы дополнительной информации (внутренние ссылки), а также ссылки на разделы сайтов, размещенных в сети Интернет (внешние ссылки). Для перехода по внешним ссылкам необходимо, чтобы компьютер имел доступ в сеть Интернет.

Для пользователей конфигураций ПРОФ-версий доступна настройка показа дополнительной информации при начале работы программы. Если дополнительная информация показывается в режиме рабочего стола, то в командной панели размещается флажок "Показывать при начале работы". Если снять его установку, то дополнительная информация не будет показываться при начале работы. Флажок "Показывать при начале работы" автоматически сбрасывается два раза в месяц и после обновления версии программы.

Для пользователей базовых версий конфигураций показ дополнительной информации при начале работы программы выполняется всегда.
