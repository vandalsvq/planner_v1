Справочник предназначен для примера использования подсистем: Групповое изменение объектов, Запрет редактирования реквизитов объектов, КонтактнаяИнформация, Присоединяемые файлы, Работа с файлами.

В подсистеме Групповое изменение объектов элементы справочника демонстрируют возможность группового изменения любых реквизитов, кроме реквизита СкрытыйРеквизит.

В подсистеме Запрет редактирования реквизитов объектов справочник показывает пример того, как реквизиты ВидНоменклатуры и Код могут быть заблокированы для изменения. В справочнике также переопределена форма для разблокирования реквизитов (форма РазблокированиеРеквизитов).

В подсистеме Контактная информация форма элемента справочника демонстрирует выбор страны происхождения товара из справочника СтраныМира и автоматическое создание элементов СтранМира по данным классификатора ОКСМ при выборе страны происхождения товара автоподбором.

В подсистеме Присоединяемые файлы элементы справочника используются в качестве объектов-владельцев файлов вложений. По команде "Перенести файлы" можно перенести файлы из подсистемы Работа с файлами в подсистему Присоединенные файлы (обработка "Демо: Перенос файлов"). В форме элемента приведен пример отображения картинки из присоединенного файла.

В подсистеме Работа с файлами элементы справочника используются в качестве объектов-владельцев файлов вложений.
