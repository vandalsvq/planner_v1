Предназначен для управления файлами, присоединенными к объекту программы.

Открывается по команде Присоединенные файлы панели навигации объекта программы. С помощью соответствующих команд в список можно добавлять новые файлы, копировать, изменять или удалять их.

Зеленым цветом отображаются файлы, занятые Вами для редактирования, а серым цветом - файлы, занятые другими пользователями программы.

В списке выводится:

  •  - подписан ЭП.
  •  - зашифрован.
  • Наименование - полное наименование файла. Значок рядом с наименованием указывает тип файла.
  • Описание - подробное описание файла.
  • Дата изменения - дата изменения данных версии файла (местное время);
  • Автор - пользователь, который добавил версию файла;
  • Размер (КБ) - размер данных версии файла;

С помощью команды Все действия - Изменить форму можно добавить сведения:

  • Расширение - формат файла, рядом значок, который также показывает формат;
  • Редактирует - пользователь, который занял файл для редактирования и т.д. 

Загрузка файлов

  • С помощью команды Все действия - Загрузить файлы можно разместить в списке файлы с жесткого или сетевого диска.
    • В открывшемся Проводнике Windows укажите путь к файлу, нажмите Открыть.
    • Заполните необходимые для загрузки сведения.
  • Также можно перетащить один или несколько файлов из Проводника Windows с помощью мыши. Если был добавлен один файл, автоматически создается и открывается карточка файла. При добавлении нескольких файлов карточки файлов автоматически создаются, но не открываются.
