Список предназначен для управления файлами программы.

В списке выводятся сведения:

  •  - подписан ЭП;
  •  - зашифрован;
  • Наименование - полное наименование файла. Значок рядом с наименованием указывает тип файла;
  • Владелец - папка списка файлов или объект программы, к которому присоединен файл;
  • Автор - пользователь, который добавил файл;
  • Создан - время и дата добавления файла в базу;
  • Расширение - расширение файла;
  • Размер (Кб) - размер файла;
  • Редактирует - пользователь, который занял файл для редактирования;
  • Описание - подробное описание файла;
  • Версия - номер версии файла;
  • Отредактировал - пользователь, который добавил версию файла.
  • Том - том на диске, в котором хранятся файлы;
  • Путь к файлу на томе;
  • Дата - дата добавления версии файла в базу.

Доступные действия:

  • Просмотреть - Открыть файл внешним приложением для просмотра;
  • Отменить редактирование - снять с файла пометку занятости для редактирования;
  •  Изменить - открыть карточку файла;
  • Сохранить как - сохранить файл в указанный каталог на жестком или сетевом диске.
  • Удалить (Shift + Del) - удалить файл окончательно.
