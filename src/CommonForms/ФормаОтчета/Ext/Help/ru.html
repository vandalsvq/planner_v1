Для просмотра отчета необходимо нажать кнопку Сформировать, предварительно задав параметры (если они предусмотрены). В клиент-серверном варианте работы программы предусмотрено формирование отчетов в фоновом режиме.

Просмотр отчета

  • По умолчанию форма отчета состоит из двух частей: в левой части выводятся сведения отчета, в правой части располагаются быстрые настройки. 
  • Для удобства просмотра отчета предусмотрена кнопка Настройки. Нажмите кнопку для того чтобы скрыть быстрые настройки и увидеть весь отчет. Повторное нажатие кнопки возвращает настройки.
  • В программе может быть предусмотрено несколько вариантов отчетов, выполняющих схожий анализ. В этом случае в программе предусмотрены:
    • для настройки отчетов - вкладка Настройки,
    • для просмотра списка других вариантов отчетов - вкладка Другие отчеты. Кнопка Настройки приобретает вид Настройки и другие отчеты, нажав ее, можно показать или скрыть обе вкладки.

Настройки

  • Основные параметры отчета выведены на вкладку Настройки, например, Период. Период можно выбрать из списка:
    • год (месяц, день) - в этом случае можно выбрать нужное значение из списка (Фиксированный) или выбрать Относительный и в качестве значения указать Прошлый, Этот и т.п.
    • произвольный период, в этом случае укажите даты начала и окончания периода. 
  • Для отбора данных по нужным параметрам включите соответствующие флажки.
  • Настройки сохраняются.
  • По команде Все действия - Все настройки можно перейти к настройке всех предусмотренных параметров отчета. 
  • Предусмотрена настройка отборов:
    • единичных (используются операции сравнения Равно, Не равно, В группе, Не в группе). После выбора операции укажите значение для отбора.
    •  множественных (операции сравнения В списке, Не в списке, В группе из списка, Не в группе из списка). Для множественных отборов нажмите кнопку Подбор, выберите значения из предлагаемого списка.
  • Укажите необходимые параметры, нажмите кнопку Завершить редактирование. 
  • Нажмите Все действия - Сохранить настройки, для того чтобы записать текущие настройки отчета.
  • Нажмите Все действия - Загрузить настройки, для того чтобы воспользоваться ранее сохраненными настройками отчета.

Другие отчеты

На вкладке Другие отчеты размещен список похожих по смыслу отчетов. 

  • В списке выводятся наименования и описания отчетов.
  • Список отчетов сгруппирован. Группа Варианты отчета предназначена для быстрого выбора других вариантов этого отчета.
  • Могут быть другие группы, в которых выводятся схожие по смыслу другие отчеты той же подсистемы. 
  • Для перехода к другому варианту отчета нажмите на заголовок левой кнопкой мыши.

Печать отчета

  • Сформированный отчет можно распечатать, нажав кнопку Печать.
  • Для того чтобы проверить отчет и изменить параметры страницы, нажмите Предварительный просмотр.
  • С помощью команды Все действия - Параметры страницы можно настроить печать.

Отправка отчета по электронной почте

С помощью команды Отправить отчет можно:

  • Отправить сейчас, для этого:
    • с помощью флажков выберите один или несколько форматов, в которых необходимо отправить отчет.
    • при необходимости включите флажок Дополнительно упаковать в архив (.zip).
    • Нажмите Продолжить. Отчет будет прикреплен к электронному письму в качестве вложения.
  • Создать рассылку отчетов;
  • Включить в существующую рассылку отчетов. 

Выбор варианта отчета

  • Для загрузки ранее сохраненного варианта отчета нажмите кнопку Выбрать вариант или воспользуйтесь командой Все действия - Выбрать вариант.
  • Нажмите кнопку Выбрать для перехода к нужному варианту отчета.

Изменение варианта отчета

  • Нажмите Все действия - изменить вариант отчета. Можно добавить поля, изменить параметры и настройки отчета.
  • Внесенные изменения можно сохранить в другой вариант отчета.

Сохранение варианта отчета

  • Нажмите Все действия - Сохранить вариант, укажите Наименование и другие параметры.

Сброс настроек

Нажмите Все действия - Сбросить настройки, для того чтобы сбросить пользовательские настройки.

Другие доступные действия

  • Для поиска в отчете воспользуйтесь командой Найти.
  • С помощью кнопок  Свернуть все группы или  Развернуть все группы можно скрыть или показать группировку отчета.
  • Для сохранения результата отчета в файл нажмите .
  • Нажмите  для того чтобы посчитать сумму чисел в выделенных ячейках.
